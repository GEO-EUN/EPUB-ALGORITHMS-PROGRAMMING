선택 구조

선택 구조는 주어진 조건에 따라 실행할 명령이 달라지는 제어 구조입니다. '만약 ~라면 ~하고, 아니면 ~한다'와 같은 형태로, 조건의 참/거짓에 따라 프로그램의 흐름이 다른 경로로 진행됩니다. 주로 if-else 문으로 구현됩니다.

🎯 직접 입력해보기

💡 학습 팁
  • 다양한 점수와 기준점을 설정하여 선택 구조를 이해해 보세요
  • 경계값(기준점 근처)을 테스트해 보세요
  • 기준점을 바꿔가며 결과 변화를 관찰해 보세요
  • 알고리즘 실행 시뮬레이션에서 단계별로 확인해 보세요

📋 미리 준비된 테스트 케이스

일반적인 합격: 점수=85, 기준=60
결과: 합격
일반적인 불합격: 점수=45, 기준=60
결과: 불합격
경계값 테스트: 점수=60, 기준=60
결과: 합격
경계값 테스트: 점수=59, 기준=60
결과: 불합격
높은 기준: 점수=95, 기준=90
결과: 합격

📊 실행 결과 비교

알고리즘 실행 시뮬레이션

1 시작
2 점수 입력:
3 점수 >= 60 인가? ?
4a ✅ 참: "합격" 출력
4b ❌ 거짓: "불합격" 출력
5 결과: ?
6 종료

변수 상태

점수: - 결과: -
예시 코드 보기 
START
점수 입력
IF 점수 >= 60 THEN
  "합격" 출력
ELSE
  "불합격" 출력
ENDIF
END
